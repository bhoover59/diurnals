# diurnals
## About
Input data frame to produce diurnal trend. Consists of 3 functions
1. DiurnalAvg()
-Parameters
 -df = DataFrame with data to get diurnal trend
 -TimeColumn = string, name of time column in data frame df
 2. DiurnalMed()
-Parameters
 -df = DataFrame with data to get diurnal trend
 -TimeColumn = string, name of time column in data frame df
 3. char_to_time()
 -Purpose: convert string to as.POSIXct
